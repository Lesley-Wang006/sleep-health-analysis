# 😴 Sleep Health Analysis

Exploratory data analysis of sleep patterns across 374 individuals.

## Key Findings
- Sales Representatives had the lowest sleep duration and quality
- Higher BMI is linked to elevated blood pressure and worse sleep
- Stress level is the strongest predictor of poor sleep quality
- Higher daily steps correlate with better sleep

## Tools
Python | pandas | matplotlib | seaborn | Jupyter Notebook

## Dataset
[Sleep Health and Lifestyle Dataset - Kaggle](https://www.kaggle.com/datasets/uom190346a/sleep-health-and-lifestyle-dataset)
